# NestJS Monorepo Example

## Description

NestJS Monorepo Example (ref. [NestJS Monorepo](https://docs.nestjs.com/cli/monorepo))

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# Run user-api
$ nest start user-api

# Run user-api (Watch)
$ nest start user-api -w

# Run admin-api
$ nest start admin-api

# Run admin-api (Watch)
$ nest start admin-api -w
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
